# 🦾 기술 소개

- Image Captioning: CLIP vs Attention => 문장 생성 결과가 더 좋았던 CLIP 사용
- Question Generation: 카카오브레인 [Pororo](https://github.com/kakaobrain/pororo) 라이브러리 사용
- Frontend: React Native, Javascript
- Backend: Spring Boot, Jpa, Fast Api
